## Kurulum

İlk olarak config.json'daki yerleri doldur.

> Örnek

```
{
  "prefix": "!",
  "sahip": "",
  "geliştiriciler": [""],
  "token": ""
}
```

Daha sonra

```sh
npm i
```
yazarak botun dosyalarını indirip

```sh
npm start
```
yazarak botu başlatın ve botu sunucunuza ekleyin.
